Distribution Functions

Console applications for distribution functions implemented in FunktionWin (see Schrausser, 2023a, 2024, res.). Theta applications generating distributions and estimators for several parameters θ within different designs via bootstrap method, with given number of resamples B, where bootstrap estimator

θ^B=B−1·Σi=1Bθi∗.

Implemented in ThetaWin (see Schrausser, 2009, 2023b, res.).

Binomial

f(X≤k|n)=Σi=0kn!i!·(n−i)!·pi·q(n−i).

Usage:

 Binomial [p] [k] [n] [[1]]
 [p] ............. probability of event A
 [k] ............. n of events A
 [n] ..............n of trials
 [1] ............. (1) full output

Binomial_T

f(X<b|b,c)=p=Σi=0b(b+c)!i!·(b+c−i)!·2−i·2−(b+c−i).

Usage:

 Binomial_T [b] [c] [[1]]
 [b] ............. cell count b
 [c] ............. cell count c
 [1] ............. (1) full output

epsilon

ε=μ1−μ0σ^.

Usage:

 epsilon [mode] [Q0] [s] [n] [e|Q1] [p] [df] [[x]]
 [mode] .......... (1)Effect-size (2)Theta.1
 [Q0] .............Theta.0
 [s] ............. Standard deviation
 [n] ............. n of cases
 [e|Q1] .......... Epsilon | Theta.1
 [p] ............. Percent-level (0.00)
 [df] ............ Degrees of freedom n - (.)
 [x] ............. Test value

F_Function

F(F,df1,df2)=1−pα2=∫0FΓ(df1+df22)Γ(df12)·Γ(df22)·(df1df2)df12ΓFdf12−1·(1+df1df2·F)−df1+df22 dF.

Usage:

  F_Function [mode] [x] [n1] [n2]
  [mode] ......... (1)Fx=p->F (2)Fy=F->p
  [x] ............ p-Wert/F-Wert
  [n1] ........... n1
  [n2] ........... n2

Fisher_Exact

f(X=a|a,b,c,d)=P0=(a+b)!⋅(c+d)!·(a+c)!·(b+d)!(a+b+c+d)!·a!·b!·c!·d!,

f(X≤n|a,b,c,d)=pexact2=Σi=0nPi; Pi≤P0.

Usage:

  Fisher_Exact [a] [b] [c] [d] [[1]]
  [a][b][c][d] ... cell counts a,b,c,d
  [1] ............ (1) full output

Fisher_Z

Z=12·loge(1+r1−r),

r=e2·Z−1e2·Z+1.

Usage:

  Fisher_Z [mode] [x]
  [mode] ......... (1)r->Z (2)Z->r
  [x] ............ r-value/Z-value

GAMMA_Function

f(x,t)=Γ=∫0∞tx−1·e−t dt+c.

Usage:

  GAMMA_Function [mode] [value]
  [mode] ......... (1)F(x)->⌈ (2)F'(⌈)->x
  [value] ........ x / ⌈

Hypergeometric

f(X≤k|n,K,N)=Σi=0k(Ki)·(N−Kn−i)(Nn).

Usage:

  Hypergeometric [k] [n] [N] [K] [[1]]
  [k] ............ n of events A in Sub-Population
  [n] ............ size of Sub-Population
  [N] ............ size of Population
  [K] ............ n of events A in Population
  [1] ............ (1) full output

  [p] ............ probability of event A

t_Function

F(t,df)=p=∫−∞tΓ(df−12)Γ(df2)·(df·π)−12⋅(1+t2df)−df−12 dt.

Usage:

  t_Function [mode] [x] [n]
  [mode] ......... (1)Fx=p->t (2)Fy=t->p
  [x] ............ p-value/t-value
  [n] ............ n of cases

x2_Function

F(χ2,df)=1−pα2=∫0χ212df2⋅Γ(df2)·(χ2)df2−1·e−χ22 dχ2.

Usage:

  x2_Function [mode] [x] [n]
  [mode] ......... (1)Fx=p->x² (2)Fy=x²->p
  [x] ............ p-value/x²-value
  [n] ............ n of cases

z_Dichte

f(z)=ϑ=12·π·e−z22,

f−1(z)=f(ϑ)=z=ln⁡(ϑ(2·π)−1)−2,

F(z)=p=∫−∞zϑ dz.

Usage:

  z_Dichte [modus] [wert] [[f]]
  [modus] ........ (1)fx=z->d (2)fy=d->z (3) ∫x=z->p
                   (4) ∫'p->z (5) ∫'p->d (6) ∫y=d->p
  [wert] ......... z-Wert/z-Dichte/Prozentrang p
  [f] ............ (1)z-Dichte Funktionsgraph
